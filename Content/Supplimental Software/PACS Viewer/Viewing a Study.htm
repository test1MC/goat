Screen Layouts and View Modes

Screen Layouts and View Modes are access by pressing the Screen Layout button in the tool-bar. Screen layouts are grouped into four categories:

  • Image: These layouts display one or more images from a single series and are also referred to image tile layouts. The dotted lines represent images within the same series.
  • Series: These layouts display more than one series from a single study. The grey lines represent series within a single study.
  • MPR / 3D (if enabled): These layouts are available if the MPR/3D feature is enabled, and display various reformatted series stacks (MPR, MIP or 3D Volume). See section "Advanced Visualization (MPR/3D) View Modes" for more details.
  • Study Compare (if enabled): These layouts control layout behavior when prior studies are in-view in single or multi-monitor modes.

MPR / 3D View Mode

If the 3D module is enabled in eUnity, and the Site Administrator has enabled this option, the user will see a new set of Layout options between the Series and Study Compare sections.

Note: All reformatted and volume rendered views (MPR and 3D) are treated as 'Lossy' and will display a 'Lossy' Indicator in the upper-right corner of the view.

Interactive Mode

Interactive mode is a special low-latency mode that prioritizes frame-rate and is enabled in 'fast' operations such as live/on-the-fly Window/Level adjust, Stack navigate or adjusting the transfer functions or volume orientation on volume rendered stacks.

Note: View-ports in "Interactive mode" will display a spinner in the upper-right corner of the view-port denoting higher quality images are in progress.

Advanced Visualization Options

Advanced visualization view modes are accessible by pressing the "Screen Layout" button and selecting the desired view mode from the MPR/3D section.

  • Basic MPR (MPR) View Modes: These layouts will generate axial, sagittal and coronal planes from the selected series and display it in either a 2x3 or 3x2 layout format. Slice navigation in this mode is similar to 2D stack navigation (e.g. mouse wheel and drag advances by the slice thickness).
  • Advanced MPR (ADV) View Modes: These layouts will generate axial, sagittal and coronal planes from the selected series, but also provide advanced features like ability to change slice thickness, MIP rendering type, and creating oblique/non-orthogonal stacks.

Changing Slice Thickness

When a user hovers their mouse over the image, a slice thickness slider will display on the right-hand side of the view-port. Using the slider will allow the user the ability to control the slice thickness of all reformatted stacks.

Changing Rendering Mode

When a user hovers their mouse over the image, a rendering mode drop-down control will display in the upper-left hand corner of the view-port (see image to the right). Click the "+" button to render the reformatted stacks as Maximum Intensity Projection (MIP), Minimum Intensity Projection (MinIP), or Average Intensity Projection (Average).

Create non-orthogonal/oblique reformatted stacks

Every view-port is annotated with a color that directly corresponds to the reference line on other intersecting images in other view-ports. To generate new planes, drag the controls corresponding to the color of the view-port you wish to change until desired orientation is achieved.

Advanced MPR and Volume Rendered (3D) View Modes

These layouts will generate ADV stacks (providing the same options) but also display a view-port with a 3D volume rendered stack.

The following features are enabled on the volume-rendered series:

Adjust the Rendering Presets

In the upper-left corner of the screen, below the 'Final Quality' label, is a list of configured rendering presets. This list can be adjusted on-site and may differ from the default. Changing the presets will re-render the volume, highlighting different features, or even completely changing the method or type of rendering.

Adjust Interactive and Final Quality

In the upper-left corner of the screen are two controls, Interactive Quality, and Final Quality, changing these options will scale up or scale down the quality of the "interactive" images; such as images generated during on-the-fly rendering, and "final" quality, such as image generated at rest, when no user commands are applied to the volume stack.

Adjust the Transfer Function

The transfer function maps a voxel’s Hounsfield value (or other dataset unit) to three components: color, lighting and opacity values. You can adjust the transfer function by selecting the Window/Level tool then click-dragging the cursor (or dragging the finger in touch-mode) up/down and left/right on the view-port. The rendered volume will then hide or highlight certain kinds of features based on the selected preset.
