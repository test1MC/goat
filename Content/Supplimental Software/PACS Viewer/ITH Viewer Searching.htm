InTouch Viewer Searching

To find studies using InTouch Imaging search, enter applicable criteria in any combination of the available search fields:

  • Date Range,
  • Patient Name,
  • Patient ID,
  • Accession Number,
  • Modality.

The following search fields can also be configured by your site administrator:

  • Referring Physician
  • Patient Birth Date

The results present Patient and Study information within the search results area. If there are multiple pages of information users can page through results using the Previous and Next buttons at the bottom of the screen.  The default page sizes are set to 150.

The Date Range field can take a single day or two dates separated by a hyphen.  The accepted date format is DD/MM/YYYY.  Invalid formats will turn the box red.  Alternatively, one can click the date range field and select a pre-configured default or choose a range using the date picker.

If needed, you may use the * (asterisk) wildcard character in search fields to broaden search results. For example, entering 123*5 in the Patient ID field returns all patients whose ID begins with 123 and end in 5. The wildcard character can be at the start or middle of the string.

Note: The Search does automatically place an asterisk at the end of patient name and patient id searches.

Column Sort

Clicking the column header will sort the results by that field. A caret next to the header will denote which column the results are sorted by. Caret down denotes descending sort. Caret up denotes ascending sort.

Hiding/Pinning Side-Panel

The side-panel, which includes search fields and saved search templates, can be hidden or pinned. This is done by toggling the 'pin' button.

Light / Dark Search List Themes

Click the icon, located in the top-right hand corner, to toggle the search list theme from dark to light and vice versa.
