Keyboard Shortcuts

The following are default keyboard shortcuts are available in .

Note: These default may vary depending on the group you are part of, or may have been redefined by the site administrator.

Modes

Key Description
W Toggle Window-Level Mode
Z Toggle Pan/Zoom-Level Mode
N Toggle Navigate Mode
T Toggle Triangulation/Cursor Mode
L Toggle Linear Measurement Mode
E Toggle Elliptical ROI Measurement Mode
F Toggle Free-form ROI Measurement Mode
A Toggle Angle Measurement Mode
B Toggle Cobb Angle Measurement Mode
C Toggle Circle Measurement Mode

Image Navigation Tools

Key Description
Cursor Key Up Scrolls images from a higher number image to a lower number image.
Cursor Key Down Scrolls images from a lower number image to a higher number image.
Home Returns the active View Box to the first image (lowest number image)
End Returns the active View Box to the last image (highest number image)

Series Navigation Tools

Key Description
Cursor Key Left Place the previous series in the active View Box.
Cursor Key Right Place the next series in the active View Box.

Study Tools

Key Description
O Revert study to original Window-Level and remove all markup.
P Toggle Cine Mode
S Toggle Reference Lines
K Toggle Link Series
R Show Report (if available)
M Toggle Show/Hide Markup
SPACE

Toggle Cine Play/Pause
Ctrl+SPACE Capture Key Image

Window-Level Presets

Key Description
0 (zero) Revert study to original Window-Level
1 ... 9 Apply Window-Level preset.

Zoom Tools

Key Description
+ Zoom in on the active View Box.
- Apply Window-Level preset.

Markup

Key Description
Delete Deletes the selected markup.
