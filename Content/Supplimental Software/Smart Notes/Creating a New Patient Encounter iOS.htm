Creating a New Patient Encounter iOS

  1. Under the Start New Encounter section on the navigation side-bar menu, click on the StrokeRESPOND hyperlink.

NOTE: The following example will show the process for StrokeRESPOND. Other applications will have a different template, but a similar workflow. Your StrokeRESPOND template may differ from the template shown in this user manual.

  1. Select StrokeRESPOND from the new patient’s application drop-down list.
  2. Select the patient’s location from the new patient’s location drop-down list.
  3. Tap Create New Patient.
  4. Enter all required fields (*).
  5. Tap the Create button.

  • The patient encounter will be shown.
