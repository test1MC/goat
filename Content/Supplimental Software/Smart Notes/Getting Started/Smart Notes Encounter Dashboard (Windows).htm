Smart Notes - Encounter Dashboard

From the Encounter Dashboard you can create a new encounter, view and edit an existing encounter. Your account role determines the level of access you have within the application and which patients you can see. For more information on user roles -

See “Smart Notes User Roles" on page 1”.

TIP: To return to the Encounter Dashboard anytime, click the logo located at the top left corner of the screen.

Searching for an Encounter

The initial display of the dashboard displays the past 24 hours of encounters starting with the most recent encounter. To search for an encounter:

  1. Enter the last or first name of the patient in the Search dialog field.
    • Results are displayed based on entered search criteria.
  2. Click on the encounter to access.
