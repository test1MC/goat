Smart Notes - Field Types

The following field types are found throughout for users to enter data related to a patient's encounter.

Free Text

Users can type a limited number of alpha numerical characters into this field type (e.g. Name, Clinical Presentation, MRN).

Text Area

User can type unlimited alpha numerical characters in this field. Text Area can expand horizontally by clicking and dragging the lower, right hand corner.

Type Ahead

The Type Ahead control auto fills text based on the selection service attached to it (For example, Medical History, Medications, ICD-10, CPT). This control is most commonly used to auto fill selections and allows for faster input as well as error prevention.

Single-Select Radio Button

A radio button allows the user to choose only one of a predefined set of mutually exclusive options (For example, Known or Unknown).

Checkboxes

Checkboxes are on/off switches that can be toggled by users, and allow users to select one or several values for the same property (For example, Consent, Inclusion/Exclusion Criteria, Review of System).

Date and Time Stamp

This field has a formatted input mask which uses characters or symbols to define the input of the day, month, year, and time resulting in improved data integrity. Clicking the Now button timestamps that field with the current date and time in the patient’s local time.

Date Stamp

This field has a formatted input mask which uses characters or symbols to define the input of the day, month and year, and assists the user with the input of the date and improves data integrity.

Reference Field

A reference field displays a field from another part of the application. For example, Last Known Well type from the Patient Info page is referenced on the Recommendations page. Once you define a reference field, the system creates a relationship to display the referenced field in another page of the application.

Conditional Visibility

This control enables the visibility or hiding of another field on-the-fly when certain conditions are met. For example, when Last Known Well Time is ‘Known’, the ‘Time’ stamp become visible along with ‘Type’ and other questions pertaining to the patients stroke symptoms.

Automatic Unit Conversion Field

Some fields have built-in funtionality to convert from one form of measurement to another by clicking Units. For example, enter the weight in pounds, click Units and the weight is converted to kilograms.
