Smart Notes User Roles

InTouch Smart Notes has four different user roles:

  • Account Admin - Admin privileges across an entire account
  • Location Admin - Location Admin has access only to locations assigned to the users’ access map as approved by the Account Admin.
  • Provider - Used by any Healthcare professional (e.g. Physician, Nurse, Medical Assistant) that needs access to encounters. Users are granted permissions to access encounters based on their Locations access map as approved by an Account Admin.
  • Read Only - Utilized by transfer centers, medical records, billing and other support healthcare professionals.
