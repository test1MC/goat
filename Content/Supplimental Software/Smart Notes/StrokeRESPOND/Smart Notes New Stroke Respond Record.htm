Smart Notes - Creating a New Encounter

When creating a new encounter the provider is required to enter data about the patient.

  1. Enter all required fields (*):
    • Last name
    • First name
  2. Enter the following fields if available:
    • MRN - the Medical Record Number
    • Date of birth
  3. Click Create

Once the Create button is clicked, the encounter pages are displayed and the encounter is registered on the Encounter Dashboard. See "Smart Notes-Viewing and Editing StrokeRESPOND Encounters" on page 1 for more details.
