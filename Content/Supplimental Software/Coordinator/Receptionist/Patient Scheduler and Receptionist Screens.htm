Patient Scheduler and Reception Tabs - Screens

Selection Bar

Patient Scheduler- Used for:

  • creating new patient appointments,
  • adding new patients,
  • rescheduling appointments,
  • creating follow up appointments,
  • canceling appointments.

Reception - Used for:

  • checking in patients,
  • printing patient labels,
  • viewing patient information.

Provider Scheduler - Used for an Overview of different types of provider schedules and making schedule edits.

Calendar - Used for viewing and exporting calendars for the providers.

Patient Scheduler - Patient Search

Patient Search and Clinic List buttons - toggle the associated dialog boxes.

Patient Search:

  • First Name - Patient's first name
  • Last Name - Patient's last name
  • Date of Birth - Patient's date of birth
  • Email Address - Patient's email address on file
  • Mobile Number - Patient's mobile phone number on file
  • Clear Form - Clears any entered search fields
  • New Patient - Opens the New Patient window
  • Search - Runs a search of the entered patient information

Providers: Lists the available providers and allows a default provider selection as well.
