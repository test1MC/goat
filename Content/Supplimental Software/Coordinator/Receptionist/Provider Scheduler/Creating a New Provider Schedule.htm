Creating a New Provider Schedule

  1. Enter information for the following fields on the New Schedule screen:
    • Schedule Name – Name given to the schedule. For example, the name of the primary provider on the schedule.
    • Provider – Name of the provider to whom the schedule pertains.
    • Service Line – The type of medical care that a provider specializes in.
    • Start Date – The date the selected schedule begins.
    • End Date – The date the selected schedule ends.
  2. Select the type of schedule from one of the following:
    • Date Range – Enter a start date and time and an end date and time.
    • Multi-Date – Select the dates and enter a start and end time for each date selected.
    • Recurring Dates – Select a start date and an end date with a start time and end time for each date in the selected date range and select the frequency of the recurrence.
  3. Click the to add the entered schedule to for review and activation.
    • The entered schedule will appear.
    • Select additional dates and times to create more times for the schedule, clicking each time to add the additional times to the schedule.

  1. Deselect Automatically Set Availability if the provider needs to review the new schedule prior to it being set and activated.
    • With Automatically Set Availability selected, the provider does not need to review the new schedule prior to the schedule being activated.

  1. Click Create Request Schedule when done
    • The schedule summary is displayed.

  1. Click Activate Schedule to activate the displayed schedule.

  1. Click Yes when prompted, Are you sure? Activate this schedule?
    • The schedule will now display a Status of Active.
