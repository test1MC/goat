View Schedules - Provider Scheduler

Upcoming Schedules screen displays details of upcoming schedules for providers assigned to the selected clinic.

  • Schedule Name – Name given to the schedule.
  • Provider – The provider's name associated with the schedule.
  • Service Line – The type of medical care that a provider specializes in.
  • Start Date – The date the selected schedule begins.
  • End Date – The date the selected schedule ends.
  • Deadline – The number of calendar days until the provider must approve the given schedule.
  • Status – The current status of the selected schedule.
  • Clone – Allows for a copy of the selected schedule to be created.

Click on a provider's name to view details of their current schedule.
