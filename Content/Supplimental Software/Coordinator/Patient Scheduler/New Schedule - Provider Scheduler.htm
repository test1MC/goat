New Schedule - Provider Scheduler

Creating a New Schedule

  1. Enter the information for the following fields on the New Schedule screen:
    • Name – Name given to the schedule.
    • Call Panel – Name of the panel of doctors or the individual doctor to whom the schedule pertains.
    • Service Line – The type of medical care that a providers specializes in.
    • Start Date – The date the selected schedule begins.
    • End Date – The date the selected schedule ends.
  2. Select the type of schedule from one of the following:
    • Date Range – Enter a start and time and an end date and time.
    • Multi-Date – Select the dates and enter a start and end time for each date selected.
    • Recurring Dates – Select a start date and an end date with a start time and end time for each date in the selected date range and the frequency of the recurrence.
  3. Click the to add the seleted schedule.
    • The entered schedule will appear.
