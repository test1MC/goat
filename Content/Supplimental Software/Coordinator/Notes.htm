Notes

The Notes window allow entry, viewing and editing of information related to the selected patient. This information is available on the following tabs:

  • Demographics - includes information about the current visit, patient's personal information, medical history and insurance information.
  • Encounter Notes - includes encounter details, assessment, plan, subjective information and objective information
  • Coding Notes - includes a section for notes for coding and code selection
  • Consult Report - includes the signed documents for the patient selected.
