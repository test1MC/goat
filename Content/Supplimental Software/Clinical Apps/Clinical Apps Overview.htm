Overview

The InTouch telemedicine documentation tool is a web-based application and an accessory of InTouch Health’s acute care telemedicine solution. InTouch   is developed with HIPAA-compliant safeguards for use in healthcare.

Intended Use

InTouch is a software system that provides workflow and clinical documentation tools for remote participants in a telehealth process.

is intended for clinical data entry, storage and transmittance to distributed telehealth network data systems. may also be used as the interface to provide remote physicians access to the InTouch PACS Viewer. The system can be used as a workflow tool in many different types of clinical and telehealth use cases such as neurology, cardiology, mental health, pediatrics, and neonatology. Clinical expertise is required to interpret and provide an assessment and recommendation within the system.

Patient-centric workflows that integrate with hospital EHRs for seamless consults

This next generation solution delivers customizable patient-centric applications that can be integrated with hospital EHR systems, allowing patient records to be available when and where they are needed.
