Adding Addendums

After signing the Consult Note, users are redirected to the Encounter List. If users need to add an Addendum to the Signed Consult Note, the following steps need to be followed:

  1. Click on the Signed Consult Note where an Addendum will be added.

  1. Check the Add Addendum checkbox to pull up a free text box. Enter any text that needs to be appended to the current Signed Consult Note.

  1. Click Finalized to add the Addendum and re-submit the form.
