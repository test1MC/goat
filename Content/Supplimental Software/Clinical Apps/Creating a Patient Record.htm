Creating a Patient Record

Note: Creating a New Encounter will create a new patient.

Creating a New Encounter

  1. Click +New Encounter on the InTouch   Encounter List screen.

    NOTE: The fields listed will differ depending on the clinical workflow.

  2. Select the Patient’s Location from the drop-down list.
  3. Enter all required fields (*).
  4. Click Start Process—this button becomes active when all required fields have been documented.
