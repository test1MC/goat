User Interface

Encounter List

Available functions from the Clinical Apps 2.0 Encounter List Screen:

  • Create a new patient record
  • View or edit a current patient record
  • Access administrative tasks

NOTES:

  • When accessing  through a browser, do not use the back button to go to the previous screen.
  • If changes are not showing in the Encounter List screen, click the browser Refresh button.

Begin a consult by either locating the corresponding patient or by creating a new encounter by clicking + New Encounter.
