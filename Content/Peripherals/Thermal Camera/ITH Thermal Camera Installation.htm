InTouch Thermal Camera Installation

The camera must be installed to the device used for remote presence sessions.

Note: Do not install the to a PC or device that is currently being used for a remote session.

  1. Remove the camera from the box.
  2. Locate the provided USB cable.
  3. Determine where to locate the .
    • If possible, the InTouch Thermal should be placed near the device's camera for use with Dual Image during the remote session.
  4. Hold the camera gently with one hand and with the other hand, plug the small end of the USB into the camera.

  1. Plug the other end of the USB cable into an available USB port on the device or PC that is using the InTouch Viewpoint software.

  1. Restart the Viewpoint software for the InTouch Camera to become available for the next remote session.
    • See the device or Viewpoint User Guide for instructions on restarting the software or device.
