InTouch Vantage

Main Focused

Select Main focused mode to allow the remote user to focus in on a specific conversation through the Patient Access Device. Ambient noise around the Patient Access Device is filtered out so that the Provider Access Software user can clearly hear the remote person that is in their direct line of sight.

Main Immersive (InTouch Vantage V1 Only)

Select Main immersive to enable the remote user to determine where sound is coming from. It is like using your own two ears, if someone standing behind you starts talking, you can tell their position relative to yourself.

 

The audio around the InTouch Vantage is received by a pair of stereo microphones located much like ears on the InTouch Vantage head.

Boom

Select Boom to enable audio from the InTouch Vantage boom.

Line in

When selected, an auxiliary audio input can be mixed into the selected audio input.
