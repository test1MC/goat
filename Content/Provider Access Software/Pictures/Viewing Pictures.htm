Viewing Pictures

All pictures remain in memory until the end of a session and then are discarded unless the image is saved.

  • To view saved images from the current or a previous session, or to import a media file from another application, click the . Browse to the file location, if required, and open the desired image.
  • To view pictures taken during a session but not saved, click on the Media Browser or the Thumbnail slider to scroll through the media.
  • Drag the Thumbnail slider to see thumbnail views of all the stored media.

The Media Counter displays information about media (images and videos) which have been captured or reviewed (previously saved and opened for reviewing) during the current session (e.g., 3 of 5). (See “Media Counter” for more information.)

NOTE: The Provider Access Software can load any picture stored in JPEG, TIFF, or BMP format on local machine. A picture saved in .ithimg format (encrypted) can only be viewed by the user that saved it and can only be viewed within the Provider Access Software.
