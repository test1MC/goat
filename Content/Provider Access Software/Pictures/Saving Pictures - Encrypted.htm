Saving Pictures - Encrypted

  1. View the picture to be saved in the Remote Camera View using the Media Browser buttons.

To view saved images from the current or a previous session, or to import a media file from another application, click the . Browse to the file location, if required, and open the desired image.

  1. Click the Save icon.
  2. Name the picture and choose a storage location.
  • You can save your media to your default InTouch Media Vault, removable media, or a network drive.
  • By default, the picture will be stored as an encrypted file (.ithimg), viewable only in the Provider Access Software by the user that saved it. We recommend saving all media in the My InTouch Media Vault to keep it secure.

If you want to view the picture in a different application or allow a different user to view it, you can save it in an accessible location in an unencrypted JPEG format — See Saving Pictures - Unencrypted on page 1.

CAUTION: HIPAA regulation compliance requires that the storage of pictures be carefully managed. Consult the hospital’s HIPAA policy director for clarification.
