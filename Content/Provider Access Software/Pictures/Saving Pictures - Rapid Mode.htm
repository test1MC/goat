Saving Pictures - Rapid Mode

Rapid Mode can be used to automatically save all pictures taken with the Patient Access Device Camera during a session.

  1. Right click the Remote Controls Camera icon
  2. The Rapid Mode Setup dialog box will launch. Create a file name prefix and click OK.

All pictures taken while this mode is enabled are automatically encrypted but will be saved to the last folder location any media has been saved to.

NOTE: In some cases, an administrator may wish to restrict all storage of pictures or videos from within the Provider Access Software. Contact your account manager for assistance.
