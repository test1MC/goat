Full-screen Video Interface

The Provider Access Software Full-screen Video Interface maximizes the video to fit the entire display screen. There are no Dashboard controls, however, the mouse can still be used to control the Patient Access Device.

  • To switch to the Full-screen Video interface, click Full-screen Video.
  • To return to the Normal Interface, press the Esc key.

NOTE: To return to the Normal Interface if you are using a touch-based system, click Normal Interface in the top-right corner of the screen.
