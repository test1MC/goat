Dual View Interface

The Provider Access Software Dual View Interface provides for two remote camera views. This interface retains some of the same Advanced Controls and Local Controls functionality as the normal interface, however several buttons and panels do not appear:

  • The battery meter is not shown.
  • The only Local Controls are Live and the microphone controls.
  • The mouse can be used to control the Patient Access Device depending on the functionality of the selected camera (highlighted in yellow on the top and bottom of the camera view).
  • The Provider Access Software display resolution must be set to 1280 x 800 or greater for the Dual View to work.

To switch to the Dual View interface, click Dual View, or click on the Aux Video tab of the Advanced Controls and select the view you would like to see on each side of the Provider Access Software dashboard.

To return back to the normal interface click Normal interface or press the Esc key.

NOTE: You can record both video feeds in Dual Screen view. Press REC to begin recording.
