Alternate Interface

The Provider Access Software Alternate Interface displays a much larger Advanced Controls area. This interface contains much of the same functionality as the normal interface, however several buttons and panels do not appear:

  • The battery meter is not shown.
  • The only Remote Controls are:

    • Live, Snapshot, and Video Capture.

  • The only Local Controls are:
    • Live and the microphone controls.
    • To switch to the Alternate Interface, click Alternate Interface.

The RETURN button switches back to the normal interface. Also, you can click Normal interface or press the Esc key.
