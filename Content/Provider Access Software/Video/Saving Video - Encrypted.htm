Saving Video - Encrypted

NOTE: Videos cannot be saved while recording is in progress.

  1. View the video to be saved in the Remote Camera View using the Media Browser buttons.

To view saved videos from the current or a previous session, or to import a media file from another application, click . Browse to the file location, if required, and open the desired video.

  1. Click the Save icon.
  1. Name the video and choose a storage location. You can save your media to your default InTouch Media Vault, removable media, or a network drive.

By default, the video will be stored as an encrypted file (.ithmov), viewable only in the Provider Access Software by the user that saved it. We recommend saving all media in the My InTouch Media Vault to keep it secure.

 

If you want to view the video in a different application or allow a different user to view it, you can save it in an accessible location in an unencrypted format — See “Saving Video - Unencrypted on page 1”.

CAUTION: HIPAA regulation compliance requires that the storage of video be carefully managed. Consult the hospital’s HIPAA policy director for clarification.
