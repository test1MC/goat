Recommended Desktop Configurations

NOTE: Hardware setup may vary from the figure.

 

Camera Captures local view for display on Patient Access Device.
Computer/Touch-Screen Computer (may have touch screen capabilities).
Headset Headphones with a microphone are highly recommended.
Keyboard Used for data input and control of system features.
Microphone Captures local audio for playback on Patient Access Device.
Mouse Used to control movement of the Patient Access Device and system features.
Secondary Monitor

Dual monitor systems display the Provider Access Software on the Main monitor and can display other data (e.g., hospital EMR data) on the secondary monitor.

The Provider Access Software windowed mode allows you to move the Provider Access Software between monitors. Each time the Provider Access Software is launched, it will open on the monitor that it was previously displayed on.

Speakers Play remote audio.
