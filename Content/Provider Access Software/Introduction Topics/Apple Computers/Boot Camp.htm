Boot Camp

  1. Verify that you are connected to the Internet and that your computer is connected to an external power supply.
  2. Log in to your administrator account.
  3. Quit all open applications, and log out any other users on your computer.
  4. Verify that your Apple OS is updated to Version 10.10 (Yosemite) or newer OS X by selecting About This Mac from the Apple menu at the top-left corner of your screen. If your computer is not currently running Version 10.10 (Yosemite) or newer OS X, download the software upgrade from the Apple App Store and download all other available updates by choosing Software Update from the Apple menu.
  5. Recommended: Back up your hard drive to an external drive.
  6. Open the Utilities folder located in Applications and double click on the Boot Camp Assistant. The Boot Camp Assistant will guide you through the process of downloading the latest Windows support software, partitioning your hard drive, and installing Windows 7. Click Continue.
    • Check the boxes for both Download the latest Windows support software from Apple and Install Windows 7. Click Continue.

    • Select the option to Save a copy to an external drive.
    • Verify that your USB flash drive/external drive has a minimum storage capacity of 8 GB, that it is empty and that it has been formatted as MS-DOS (FAT).
    • Format a disk, using the Disk Utility in the Utilities folder.
    • Click Continue.
    • Choose your external drive from the drop-down list.
    • Click Save.

  1. Create a Partition for Windows.
    • Click and drag the divider to set the size of the Windows partition; a minimum of 30 GB is recommended.
    • Insert the Windows 7 installation disk
    • Click Install. The Boot Camp Assistant will create the Windows partition.
    • Restart your computer, and open the Windows 7 Installer.
    • Follow the prompts to begin installing Windows 7.
  2. Choose a location to install WindowsWhen prompted.
  3. Select the BOOT CAMP partition.
  4. Ignore the message that 'Windows cannot be installed on Disk”.
  5. Click Drive options (advanced),
    1. Click Format,
    2. Click OK.
    3. Click Next.

Windows 7 will be installed on the identified partition. Follow the prompts as needed. After it is installed the computer will automatically reboot into Windows.

CAUTION: Do not select any other partition; doing so may delete the entire contents of your Mac OS X partition.

  1. Once you have booted into Windows, install the Windows support software by opening and viewing the files on the USB flash drive/external hard drive. Open the Windows Support folder and double-click on the setup program.
  1. The Boot Camp Installer will launch. Click Next and follow the prompts. When the Boot Camp installation is complete, click Finish.
  2. Install any Windows 7 updates by going to the Windows Start button, selecting All Programs and clicking on Windows Update.

NOTE: You will need to restart your computer for the updates to take effect.

  1. Configure Boot Camp preferences by right clicking Ctrl+click Boot Camp in the System Tray and opening the Boot Camp Control Panel.
  • Select the Trackpad tab and check the Tap to Click box under the One Finger section, and the Secondary Tap (this will allow you to right click) box under the Two Fingers section. Click Apply.

  • Select the Keyboard tab and check the Use all F1, F2, etc. keys as standard function keys box. Click Apply.
  1. Install the Provider Access Software and connect a USB headset.

 

Please contact your InTouch Health representative for instructions on installing the Provider Access Software.

NOTES:

  • To boot into Windows, press and hold the option key at boot up until one or more disk images are visible. Click the Windows partition and then click on the arrow below it. Alternatively, right-click + Boot Camp in the System Tray and click Boot Camp Control Panel to set Windows as the default OS. Select the Startup Disk tab and select the BOOT CAMP partition. Click Apply.
  • To return to the Mac OS, right click the Boot Camp icon in the System Tray and click on Restart in Mac OS X.
