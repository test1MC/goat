Parallels Virtual Machine

  1. Install the Parallels Virtual Machine (version 9 or higher) and Windows 7 OS.

NOTE: Windows 7 must be used when using the Parallels Virtual Machine to run the Provider Access Software.

  1. Configure the Parallels Virtual Machine to use two CPU cores by choosing 2 from the CPUs drop-down list located in the General tab of the Windows 7 Preferences.

To open the Windows 7 Preferences when in Coherence mode choose Preferences from the Windows 7 menu in the Windows 7 toolbar. To open the Windows 7 preferences when in Window mode, click the gear icon in the lower right-hand corner of the Parallels Desktop window. To open the Windows 7 preference when in Full Screen mode open the Virtual Machine menu from the Parallels Desktop toolbar (scroll cursor to the top of the screen to make toolbar visible) and click Configure.

  1. Configure the Parallels Virtual Machine to use a minimum of 2 GB of memory (4 GB recommended) by clicking and dragging the memory size slider.

NOTE: If you receive the error message shown to the right, open the Virtual Machine menu from the Parallels Desktop toolbar and select Shut Down. Change the number of CPU cores and the size of the memory and restart the Virtual Machine.

 

  1. Configure the Parallels Virtual Machine to use your computer’s built-in camera.

 

Select the Devices menu from the Parallels Desktop toolbar. Open the External Devices or USB menu and select the Apple Built-in iSight or Apple FaceTime Camera. This must be done each time you relaunch Windows 7.

 

  1. Configure the Parallels Virtual Machine to use your USB headset.

To enable your USB headset select the Devices menu from the Parallels Desktop toolbar. Open the External Devices or USB menu and select your headset.

NOTE: It is highly recommended that you use a headset with the Provider Access Software; USB headsets have been found to work best with Apple computers.

  1. Suggested additional Parallels Virtual Machine configurations:

    • Enable Zoom Out (right click):
    • From the Apple menu open System Preferences and select Trackpad from the Hardware section. Under the Point & Click section, select Click in bottom right corner from the drop-down menu under Secondary click.
    • Start Windows automatically when Parallels Desktop is launched.
    • Open the Windows 7 Preferences and select the Options tab. In the Startup and Shutdown section, select When Parallels Desktop starts from the Start Automatically drop-down menu.
    • Optimize Windows performance.
    • Open the Windows 7 Preferences and select the Options tab. In the Optimization section check the Tune Windows for speed box.
    • Share External/USB Devices with Windows.
    • Open the Windows 7 Preferences and select the Hardware tab. In the USB & Bluetooth section check the Automatic Sharing: Web cameras box. Secondly, open the Parallels Preferences by opening the Parallels Desktop menu and selecting Preferences. In the Devices tab select. When a new external device is detected: Ask me what to do.

Disable Dynamic Resolution in Full Screen

Open the Windows 7 Preferences and select the Options tab. In the Full Screen section select Keep Ratio from the scale to fit screen drop-down list. Start your Windows virtual machine in Full Screen mode. Right click on your Windows Desktop and choose Screen Resolution. Select your desired screen resolution and restart Windows.

  1. Set the screen display to Full Screen by selecting Enter Full Screen from the View menu of the Parallels Desktop toolbar. Press Ctrl--F to exit Full Screen mode .
  2. Install the Provider Access Software on your Parallels Virtual Machine.

Please contact your InTouch Health representative for instructions on installing the Provider Access Software.

NOTE: Adobe Reader must be installed on your computer to open the Provider Access Software User Manual (located in the Help tab).

View Modes of the Parallels Virtual Machine

  • Full Screen mode (recommended)

In this mode the virtual machine’s screen is expanded to occupy the whole of your computer’s screen. You may need to adjust your screen resolution to 1280x800 or 1366x768 for best results

— See Disable Dynamic Resolution in Full Screen on page 1

For new Apple Retina displays, set the screen resolution to 1400x900 for best results.To change resolution: Go to Settings: View: Retina Display and make sure “Scaled” is checked.

  • Coherence mode

In this mode the virtual machine’s desktop is invisible allowing you to see the virtual machine’s application windows in the Mac OS X and work with them side by side with your Mac OS X applications. To launch the Provider Access Software in Coherence mode click the Parallels toolbar icon at the top of your screen and open the Windows Start Menu. Click the All Programs button, and locate and open the InTouch Health folder. Double click on InTouch CS to launch the Provider Access Software.

  • Window mode

In this mode the virtual machine screen is displayed as a separate window.

CAUTION: If using Window mode do not resize the Parallels Virtual Machine window while in a live session.
