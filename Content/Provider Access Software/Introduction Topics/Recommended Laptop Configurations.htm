Recommended Laptop Configurations

NOTE: Hardware setup may vary from the figure.

Camera Captures local view for display on Patient Access Device.
Headset Headphones with a microphone are highly recommended.
Keyboard Used for data input and control of system features.
Microphone (Location varies by model) Captures local audio for playback on Patient Access Device. (Not operational when using headset.)
Monitor Monitor displays Provider Access Software Dashboard.
Mouse Used to control movement of the Patient Access Device and system features.
Speakers (Location varies by model) Plays remote audio. (Not operational when using headset.)
