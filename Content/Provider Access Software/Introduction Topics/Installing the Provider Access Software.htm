Installing the Provider Access Software

  1. Open the Provider Access Software installer.
    • Several dialog boxes will appear and disappear. Wait for the Setup Wizard screen to appear before moving onto step 2.
  1. When the InTouch Health Provider Setup Wizard appears, click Next.
  • The End-User License Agreement will appear.
  1. Select the “I accept the terms in the License Agreement” check box , and then click Next.

  1. Click Print to print the End-User License Agreement.
  1. Click Install. The Provider Access Software installation will begin.
  2. Click Finish.

The Provider Access Software (InTouch ProviderInTouch Provider) shortcut will appear on your desktop, along with a Quick Start Guide.
