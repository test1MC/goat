Downloadable Provider Access Software

InTouch Health now offers the option to install the InTouch Health Provider Access Software yourself without the need for a technical service representative.

System Requirements:

  • Windows Operating System (OS): Windows 7 or greater
  • Mac Operating System (OS): OS X Version 10 or greater running Windows 7 in Boot Camp or Parallels Virtual Machine 10 or greater
  • Video Capture Device: Webcam
  • Audio: Speakers and a Microphone
  • Minimum Memory (RAM): 2 GB
  • Minimum Available Hard Disk Space: 1GB
  • CPU: Minimum 2 Cores
  • Bandwidth: Minimum of 764kbps up and down.
