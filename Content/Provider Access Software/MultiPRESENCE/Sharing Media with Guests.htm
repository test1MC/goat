Sharing Media with Guests

The Host and Guests are able to access the same data.

 

Once shared, Host and Guests can all telestrate on the media to point out areas of interest to each other. Each user’s default telestration color will be different. See Sharing Pictures on page 1.

For Snapshot images

Once the Host takes a snapshot, the image is automatically shared with all Guests. The image is immediately shown in telestration mode. The image is also automatically placed in the media queue of all the users—Host and Guests.

The Host shares this image with the Patient Access Device as well by clicking Share Media ; then to the prompt.

Imported images

Host opens an image ﬁle. The Host then uses the Share Media button to select to share the media with one of the following options: Both the Patient Access Device and Guests, Just the Guests, or Just the Patient Access Device. Once shared, the image is shown in telestration mode. The Host can change the sharing conﬁguration at any time by clicking the Share Media button.

Screen-captured images

Host conducts screen capture ﬁrst (See Image-grabber on page 1). The Host then uses the Share Media button to select to share the media with one of the following options: Patient Access Device and Guests, Just the Patient Access Device, or Just the Guests. Once shared, the image is shown in telestration mode. The Host can change the sharing conﬁguration at any time by clicking the Share Media button.

  • If you shift-click the Share Media icon the next media will be shared as you direct. PIP and automatic return to live view will be disabled.
