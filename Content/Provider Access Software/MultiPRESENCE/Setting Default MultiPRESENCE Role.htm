Setting Default MultiPRESENCE® Role

The Provider Access Software gives users the option to default all connections to be MultiPRESENCE connections as well as to set their default role for MultiPRESENCE connections.

  1. Click Options on the Provider Access Software dashboard to open the Options window, then select the Connectivity tab. Under the MultiPRESENCE Default Role section, select either None, Host, or Guest. The option that you choose will be your default role for all MultiPRESENCE connections.
  2. Click OK to confirm using MultiPRESENCE as the default for all your connections.

    Note: This may cause a slight degradation in non-MultiPRESENCE sessions.

  3. Click OK to close the Options dialog.
