MultiPRESENCE Overview

MultiPRESENCE is a companion product to the Remote Presence platform and has been designed specifically to enable multiple users to connect to the same Patient Access Device at the same time. MultiPRESENCE enables a group of clinical experts to collaborate together in real-time on the care of a patient.

 

In MultiPRESENCE mode, users at multiple locations using the Provider Access Software are connected to the Patient Access Device. The session is controlled by a Host user. Guests are able to join the session in progress. The Host and Guests are able to access the same data: Features included in are:

  • Independent Connections: Allows Guest users to remain in a session after the Host leaves.
  • Control Transfer: Allows the Host to transfer controls to a Guest at any time.
  • Automatic Joins: Guests automatically join a session without Host permission.
  • Waiting Room: Guests joining before the Host can speak among themselves.
  • Video Mute: All users in a can mute their video.
  • Audio Only Session: Guests do not need a web camera to join a session.
  • Peripheral Support: Guests can listen to the stethoscope or handset.
  • iOS Support: Users can access as either a Host or a Guest from their iOS devices.
  • Live video and audio
  • Snapshots taken via the Patient Access Device by the Host
  • Imported images loaded from Host’s hard drive
  • Screen-grabbed images from another application on the Host Provider Access Software (e.g. CT image accessed via PACS)

Users are able to point out areas of interest to each other in the following ways:

  • Images—ability to telestrate and see each others telestration
  • Video—ability to live cursor and see each others live cursor
