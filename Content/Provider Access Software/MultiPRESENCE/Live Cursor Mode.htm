Live Cursor Mode

Users are able to point out areas of interest to each other in the following ways:

  • Images—ability to telestrate and see each others telestration.
  • Video—ability to live cursor and see each others live cursor

During live video, if you select the Live Cursor mouse mode, your User Name is displayed next to your cursor. This is seen on everyone’s Provider Access Software (and on the Patient Access Device, if desired). See Mouse Cursor Modes on page 1.

NOTE: When in Live Cursor Mode the message, "Mirror Image", is shown on the Patient Access Device Display.

  1. Host clicks Live Cursor mode.
  2. Host selects on pop-up whether to share the Live Cursor with the Patient Access Device as well or with Guests only.
    • Host can also right-click to select to hide the live cursor from the Patient Access Device.

     

  1. Host enables a Guest’s live cursor to be shown on the Patient Access Device display by right-clicking the Guest’s thumbnail and selecting Allow the Device to hear and see this user. This enables the Guest’s live cursor to be shown on the Patient Access Device display with the Host’s live cursor.
