Getting Started

The latest Provider Access Software comes with enhanced, strong password security measures. The ﬁrst time you launch the Provider Access Software you will be required to use a username and password. You can request a User Account by visiting the InTouch Health website and clicking on the User Registration link from http://www.intouchhealth.com.

  1. Double-click the Provider Access Software icon on the desktop.

Alternately, click InTouch Provider from All Programs, InTouch Health from the Start menu.

  1. Enter your Username and Password. If you do not have an account, you can request one by clicking on the link Create New Account in the Login dialog box.

  1. Click on Login.
