Connecting to a Patient Access Device

When the Provider Access Software is first opened, the “Dashboard” interface will appear and the Connection Wizard will establish a connection with the InTouch Telehealth Network.

 

The Connection Wizard will monitor the Provider Access Software progress in connecting to the InTouch Telehealth Network. If a problem occurs, the Connection Wizard will walk you through the steps to fix your problem. For a description of the symbols used by the Connection Wizard; See “InTouch Telehealth Network Connection Wizard Symbols on page 1”.

Once you are connected, you will either see the My Care Locations list or the My RP Devices list. If the My Care Locations dialog box appears, please continue reading for instructions to establishing a connection to a Patient Access Device.

My Care Locations

  1. Click on the desired Patient Access Device name in the My Care Locations list.

Use the scroll bar to see additional Patient Access Devices or enter a Patient Access Device name or part of a name in the Search box to shorten the list. Clear the search to see the entire list again.

  1. The colored icon next to the Patient Access Device name indicates its status (see table down below).
  2. If you would like to host a MultiPRESENCE ® session, click the Host checkbox. If you would like to join a MultiPRESENCE® session as a guest, select the Guest checkbox.

  1. Double-click on the Patient Access Device name to begin a session.

Status Guide

Patient Access Device is ready. Patient Access Device is configured for a private connection, not through the InTouch Telehealth Network.
Patient Access Device is busy. Patient Access Device is Initializing.
Patient Access Device is in a MultiPRESENCE® session; You may request to join by connecting. Patient Access Device is Offline

NOTES:

  • Sessions are limited to 18 hours. To continue using the Patient Access Device, disconnect from the session and reconnect.
  • Contact your InTouch Health Representative if you would like to be migrated to My Connections.

CAUTION: A tested backup method of communication should be available in case network communication is lost.

Live Help Button: Live technical support is available through this feature 24/7.
