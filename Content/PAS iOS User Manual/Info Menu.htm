Info Menu

Tapping the Info icon () opens the Info Menu. Tap on any menu option to open it. Tap elsewhere on the Multi-Touch display to close the menu option. The Info Menu is located on the Tool Bar.

Quick Help (iPad Only)

Provides a screen overlay that describes how to use the InTouch Provider Access Software. The Quick Help screen overlay will change depending on which Tab Bar tab you are currently using (Home Help, Driving Help, Media Manager Help). This option is only available on the iPad.

User’s Guide, Live Chat, and Privacy Policy

Tapping on any of these menu options will open the associated resource in a Safari window.

To return to the InTouch Provider Access Software, open the multi-tasking bar by double-pressing the iPad / iPhone Home button and selecting the InTouch Provider Access Software.

Call Support (iPhone Only)

Tapping on Call Support will dial InTouch Health Technical Support at (877) 484-9119. This option is only available on the iPhone.

Session Info

Tap the Session Info slider to turn Session Info on. Session Info allows you to see real time network information on the Tab Display area of the InTouch Provider Access Software. This information helps to diagnose problems when contacting Technical Support and provides evaluation of network parameters during installation of Patient Access Devices or during use of the InTouch Provider Access Software. This information is only shown when in session with an Patient Access Device.

Upload Log Files

Sends local log files to InTouch Health Technical Support to aid with troubleshooting.

Connectivity

Provides information about the connectivity to the InTouch Health servers. Press and drag on the scroll bar to view additional information.
