InTouch Telehealth Network Connection Wizard

The following illustration and table describe the symbols you may see while connecting to a Patient Access Device.


A

InTouch Provider Access Software for iOS

Connection is initiated by the InTouch Provider Access Software for iOS

B

Internet

Appears translucent until connection is achieved

C

InTouch Telehealth Network Server

Appears translucent until connection is achieved

D

InTouch Patient Access Device

Appears translucent until connection is achieved

E

Green Connectivity Band

Will pulse until connection is achieved, then will appear solid between connected points. A break in the band, anywhere along its path, indicates a broken connection between points.
