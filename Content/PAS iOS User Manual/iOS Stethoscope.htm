iOS Stethoscope

The stethoscope, installed on the InTouch Vita and some other InTouch devices, allows remote auscultation from the InTouch Provider Access Software when combined with a headset such as the Sennheiser PX-200-IIi. InTouch Health recommends using only high quality headsets; request the publication  for a list of validated headsets.

Stethoscope Operation

When logged in to an Patient Access Device with a built-in stethoscope, the stethoscope button on the Toolbar will be enabled.

  1. Attach your headset to the iPad / iPhone and put your headset on.
  2. Tap the stethoscope button to enable the controls.
  3. Tap the Start button to start the stethoscope.
    • Adjust the volume using the iPad / iPhone volume buttons as needed, keeping the volume level to the lowest gain necessary to avoid sound distortions.
    • When using the stethoscope it is important that both the InTouch Provider Access Software user and the patient refrain from talking. Tap the Stop button if you need to communicate, then tap the Play button to start again.
  1. Tap the Stop button when finished.
  2. Tap the stethoscope button to disable
    the controls.

NOTE: The Patient Access Device software priority is to guarantee that sound taken in from the stethoscope is reproduced with complete fidelity on the InTouch Provider Access Software side. To achieve this, a buffering scheme is used to guard against Internet delays. This means that the audio which is heard from the stethoscope is accurate, but lagging about one second behind.

Stethoscope Technical Service

If a problem should arise in regard to use of the stethoscope on the Patient Access Device, please contact InTouch Health Technical Service, so that Technical Service can determine the source of the issue. Questions about the stethoscope controls can be answered by reference to the stethoscope manual.
