Downloading the InTouch Provider Access Software for iOS

The InTouch Provider Access Software for iOS is available for download in the Apple App store (search for InTouchHealth). All Users will need an Apple ID to download the InTouch Provider Access Software for iOS. An

Apple ID can be obtained directly from Apple at:
https://appleid.apple.com

 

The InTouch Provider Access Software is password protected and requires a username and password to access your Patient Access Devices. An InTouch Provider Access Software user account can be requested from InTouch Health at:
https://www.intouchcustomer.com/UserReg/

Once your user account has been approved by InTouch Health and the health care facility you work for, an email will be sent to you with instructions on how to create a password.

NOTE: If you are using iOS 8 or iOS 9, verify that your InTouch Provider Access Software has access to your microphone. See “No Audio from InTouch Provider Access Software on page 1” for instructions.

Downloading Updates to the InTouch Provider Access Software for iOS

When an update to the InTouch Provider Access Software is available, a number one will appear in the top right corner of the InTouch Provider Access Software icon. Visit the Apple App Store to download the software update.
