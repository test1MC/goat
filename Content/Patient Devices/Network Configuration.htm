Network Configuration

The InTouch Telehealth System is comprised of a Remote Presence Patient Access Device and a minimum of one Provider Access Software Device. The Provider Access and Patient Access Devices are linked via the Internet over a secure connection. The Patient Access Device operates on either an 802.11 ac/a/b/g/n Wi-Fi network or via Ethernet.

Wireless Network Considerations

The InTouch requires a wireless network distributed over the area where it is intended to roam. The InTouch may be configured for 802.11 ac/a/b/g/n. The optimal configuration is determined during installation.

To attain optimal performance there are several considerations which must be taken into account when designing the wireless network.

Configuring the InTouch Wireless Connection

The InTouch Control Core uses a Windows computing environment and a wireless network card. This card is an Intel Ultimate N WiFi Link 6300 (Intel Dual Band Wireless AC 7260 WiFi).

The wireless link can be configured using the Microsoft Windows Wireless configuration tool.
