Troubleshooting

Not showing on display (All )

  • Confirm the display is turned on.
  • Make sure the is powered on by verifying the LED on the is illuminated or that the is lit.
  • Following the display's manufacturer’s procedures, verify the display is set to the HDMI input that corresponds to where the is connected to the display.
  • Verify the is plugged in fully to the HDMI input. If using the HDMI extension cable, verify both ends are properly connected.

Screen does not properly fit display (All )

  • This is usually a TV settings issue and the fix is dependent on the TV's manufacturer.
  • In most instances, change the Aspect Ratio to “Just Scan” from the TV's menu or cycle through available TV Aspect Ratio options to find the most suitable one.

No Volume or Unable to adjust volume (All )

  • Volume can be controlled or adjusted using the:
    • Audio/Video wizard
    • Window's volume setting either from the desktop or using the volume controls on the keyboard
    • Settings on the display device

Speakerphone not working ( and )

  1. Reconnect speakerphone.
  2. Make sure the button at center is not flashing, which means it has muted.
  3. Rerun Audio/Video wizard.
