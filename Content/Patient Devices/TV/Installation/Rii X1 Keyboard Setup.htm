Rii X1 Keyboard Setup

These instructions are for use with the Rii X1 Bluetooth Keyboard that is included with the Kit Model version of the ™.

Slide the power switch on the top left edge of the keyboard to turn on the Bluetooth keyboard/mouse.

  • A solid green light means the keyboard is connected.
  • If the Bluetooth keyboard/mouse falls asleep or loses connectivity with the television, toggle the on/off switch and the press the Bluetooth button.
  • If you experience Bluetooth connectivity challenges, remove the USB receiver dongle from the main unit flank chamber and insert it into an available port on the InTouch TV™.
