InTouch Viewpoint Software System Requirements

Windows:

Windows Version

7, 8.1, 10

Memory
  • 4 GB RAM

    • Processor

    Intel Core i5 or above processor

    Camera

    USB HD camera

    USB Ports

    At least 1 USB port

    Video

    Intel HD Graphics Driver

    Speaker/Microphone

    Echo canceling speaker/ microphone

    Optional
    • Integrated Bluetooth adapter for Bluetooth peripherals
    • Multiple, independent USB ports for USB video sources

    • USB 3.0 port is recommended for HD imaging sources

    iOS:

    iOS Version

    10 or higher

    Memory
  • 2 GB RAM

    • Recommended Models

    iPad Pro, iPad Air, iPad Mini with front and rear HD cameras
